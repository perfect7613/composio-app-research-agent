Headline Findings

Easy wins exist, but they are not evenly distributed.

Self-serve developer platforms, ecommerce, support, and project-management apps are the fastest route. Ads, fintech, enterprise data, and some AI-native products need review or outreach before engineering should invest heavily.

Patterns

OAuth dominates customer workflows; keys dominate data utilities.

The recurring blocker is not “no API.” It is the path from public docs to usable credentials and safe actions.

Agent + Human Loop

The pipeline makes the work repeatable; humans handle ambiguity.

1

Seed

`data/apps.json` fixes the 100-app research set.

2

Research

`npm run research` uses Firecrawl to draft structured records from official docs.

3

Normalize

Auth, gatedness, API breadth, MCP, verdict, blocker, and evidence are made comparable.

4

Verify

`npm run verify:sample` uses Browser Use Cloud for repeatable spot checks.

5

Correct

Human review fixes overclaims, gated docs, and product/API ambiguity.

Where a product was gated behind payment, partnership, admin approval, or unclear docs, the correct answer is to say so with evidence.
